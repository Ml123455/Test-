🗺️ Road Trip Via Ferrata — 8 Jours

Départ Bonnelles · 6 ferratas · 🛶 Paddle/Kayak · 5 soirées club · Groupe 4-5 potes

~2800 km · 420-500€/pers · J6 repos avant le Diable 🏆

J0 🚗 Transfert — Bonnelles → Canet
🕐 Départ 18h de Bonnelles (78). 660 km via A6/A7/A9.
🛣️ ~6h30 de route. Relais conducteur toutes les 2h.
🌙 Arrivée vers minuit à Canet. Nuit camping La Marina.
⚠️ Pas de ferrata ce jour. Juste la route. Dormez.
J1 🌊 Perpignan — Plage + Château cathare
🏖️ Matin : Plage de Canet — sable fin, paddle, première baignade mer. Le trip commence les pieds dans l'eau !
🧗 Aprem : Via Ferrata du Fenouillèdes (4.8★, 1900 avis). 2 parcours : Le Panoramique (AD, 2h, ponts de singe) et La Pichona (F, initiation). Vue sur les vignes cathares.
🎉 Soir : Le Speakeasy — house music, centre Perpignan. Entrée 10€.
✅ Location matos : Speleteaux Saint-Paul. Accès libre, pas de guide.
J2 🌊 Montpellier — Kayak + Grotte
🏖️ Matin : Plage de Palavas-les-Flots — kayak de mer, 15 min de Montpellier. Dernière baignade dans le sel !
🧗 Aprem : Via Ferrata du Thaurac (AD, 2h). Gorges de l'Hérault — traversée de grotte, ambiance spéléo unique.
🎉 Soir : Panama Café — entrée GRATUITE, ambiance étudiante, 00h-6h.
✅ Location matos : Roc'N River (13€, 5min du site).
J3 Ardèche — Tyrolienne + Rivière
🛣️ Route : Montpellier → Thueyts (1h30, A75).
🧗 Matin : Via Ferrata Pont du Diable (AD, 2h). Tyrolienne de 90m au-dessus de la rivière Ardèche. Site volcanique exceptionnel.
🏊 Aprem : Baignade dans l'Ardèche, sous le pont. L'eau est fraîche, parfaite après l'effort.
🎉 Soir : L'Esclave à Avignon (1h de route). Club intra-muros, ambiance jeune, 10-15€.
✅ Location matos : Cimes-Canyons (sur place à Thueyts).
J4 Verdon — Canyon + Cordata
🛣️ Route : Avignon → Lac de Castillon (1h30).
🏊 Matin : Lac de Castillon — pédalo, paddle, kayak. Baignade dans un cadre alpin.
🧗 Aprem : Via Cordata du Trou du Renard. Pas une ferrata classique : cordes fixes, tyrolienne, rappels, sauts. Guide obligatoire (fournit le matos).
🎉 Soir : Le Mistral à Aix-en-Provence. Institution électro/techno. 20€ l'entrée.
⚠️ Via CORDATA, pas ferrata. Guide obligatoire (~50-80€). Pas d'accès libre.
J5 Serre-Ponçon → Grenoble
🛣️ Route : Aix → Lac de Serre-Ponçon (2h30, A51).
🏊🛶 Journée : Kayak et paddle sur le plus grand lac alpin de France. Eau turquoise, montagnes en toile de fond.
🛣️ Fin PM : Route Serre-Ponçon → Grenoble (2h, par Gap).
🍺 Soir : Bars chill à Grenoble — PAS de club. Louer le matos chez Takamaka (15€) pour J7 !
⚠️ Pas de ferrata ce jour. Récupération active. Louer matos à Grenoble pour le Diable J7.
J6 ☀️ REPOS — Lac + Bastille (option)
🏊🛶 Matin : Lac de Paladru — paddle tranquille, sieste sur la plage. VRAI repos.
🧗 Option : Via Ferrata de la Bastille — 45 min, urbaine, très facile. Panorama sur Grenoble. Si t'as la foi, sinon sieste.
🌙 Soir : PAS de club. Dodo tôt en F1/hôtel. Demain c'est le GRAND JOUR.
✅ Location matos Bastille : Takamaka 15€. Mais priorité = repos.
J7 🏆 LE JOUR J — Via Ferrata du Diable
🛣️ Route : Grenoble → Aussois (1h30).
🧗 LA FERRATA : 7 tronçons, 3760m cumulés, du PD au TD. La + longue de France. Gorges de l'Arc, Pont du Diable, Forts de l'Esseillon. Jambes fraîches après J6 repos !
🎯 Objectif : 3-4 tronçons PD/AD (3-4h). Les TD c'est pour les pros. Profitez, vous l'avez mérité.
🎉 Soir : CO2 Club à Chambéry — 2 salles, 2 ambiances. Dernière soirée du trip !
⚠️ Matos loué à Grenoble la veille. Pas de location à Aussois. Visez PD/AD, pas TD.
J8 🏊 Annecy — Retour
🛣️ Route : Chambéry → Annecy (40 min).
🏊🛶 Matin : Lac d'Annecy — dernière baignade, paddle, farniente. L'eau la plus pure d'Europe.
🚗 Aprem : Retour Annecy → Bonnelles (~550 km, 5h30). Arrivée dans la soirée.
✅ Fin du trip. Vous avez survécu au Diable. Respect.
