🗺️ Road Trip Via Ferrata — 8 Jours

Départ Bonnelles · 3 ferratas · Canyoning · Rafting · Rando · 3 clubs + 2 bars

~2663 km · ~565-625€/pers · Canyoning + Rafting + Rando glaciers

J0 🚗 Transfert — Bonnelles → Canet
🕐 Départ 18h de Bonnelles (78). 660 km via A6/A7/A9.
🛣️ ~6h30 de route. Relais conducteur toutes les 2h.
🌙 Arrivée vers minuit à Canet. Nuit camping La Marina.
⚠️ Pas de ferrata ce jour. Juste la route. Dormez.
J1 🌊 Perpignan — Plage + Château cathare
🏖️ Matin : Plage de Canet — sable fin, paddle, première baignade mer. Le trip commence les pieds dans l'eau !
🧗 Aprem : Via Ferrata du Fenouillèdes (4.8★, 1900 avis). 2 parcours : Le Panoramique (AD, 2h, ponts de singe) et La Pichona (F, initiation). Vue sur les vignes cathares.
🎉 Soir : Le Speakeasy — house music, centre Perpignan. Entrée 10€.
✅ Location matos : Speleteaux Saint-Paul. Accès libre, pas de guide.
J2 🌊 Montpellier — Kayak + Grotte
🏖️ Matin : Plage de Palavas-les-Flots — kayak de mer, 15 min de Montpellier. Dernière baignade dans le sel !
🧗 Aprem : Via Ferrata du Thaurac (AD, 2h). Gorges de l'Hérault — traversée de grotte, ambiance spéléo unique.
🎉 Soir : Panama Café — GRATUIT, ambiance étudiante. ⚠️ 2.9★, videurs sélectifs, jeu-sam uniquement. Plan B : Rockstore Montpellier (institution rock/électro).
✅ Location matos : Roc'N River (13€, 5min du site).
J3 Ardèche — Tyrolienne + Rivière
🛣️ Route : Montpellier → Thueyts (1h30, A75).
🧗 Matin : Via Ferrata Pont du Diable (AD, 2h). Tyrolienne de 90m au-dessus de la rivière Ardèche. Site volcanique exceptionnel.
🏊 Aprem : Baignade dans l'Ardèche, sous le pont. L'eau est fraîche, parfaite après l'effort.
🎉 Soir : L'Esclave à Avignon (1h de route). ⚠️ 3.0★, club gay-friendly, porte sélective. Si pas à l'aise → bars chill place de l'Horloge.
🌙 Nuit : Avignon F1/hôtel (~25€/pers). Impossible de rentrer à Thueyts après le club (1h de route).
⚠️ Journée intense (ferrata + route + club). Pensez à bien vous hydrater.
✅ Location matos : Cimes-Canyons (sur place à Thueyts).
J4 Verdon — Canyon + Rando Aquatique
🛣️ Route : Avignon → Lac de Castillon (1h30).
🏊 Matin : Lac de Castillon — pédalo, paddle, kayak.
🌊 Aprem : Canyoning débutant dans le Verdon — rando aquatique, toboggans naturels, vasques. 2-3h, accessible à tous. 40-50€.
🎉 Soir : Le Mistral à Aix-en-Provence. ⚠️ 2.2★/541 avis, cher (>20€), porte très sélective. Plan B : Le Divino à Aix (bar lounge, 4.2★).
⚠️ Journée intense (canyoning + route + club). Pensez à bien vous hydrater.
✅ Canyoning encadré par un moniteur. Aucun matos à prévoir, tout fourni.
J5 Serre-Ponçon — Rafting + Grenoble
🛣️ Route : Aix → Embrun (2h, A51).
🛶 Matin : Rafting sur la Durance — descente de 11km, rapides classe 2-3, débutants OK. 35€, 2h sur l'eau. De Bleu à Blanc.
🏊 Aprem : Baignade Lac de Serre-Ponçon, kayak, paddle.
🍺 Soir : Bars chill à Grenoble (1h30 de route). PAS de club. Retour au bivouac Serre-Ponçon après les bars (30 min, Sam conducteur sobre).
⚠️ Journée intense (rafting + route + bars). Pensez à bien vous hydrater.
✅ Rafting encadré, matériel fourni. Juste prévoir un maillot et une serviette.
J6 ☀️ REPOS — Lac de Paladru
🏊🛶 Journée : Lac de Paladru — paddle, sieste, farniente. Plages aménagées, eau à 24°C. VRAI repos.
🍽️ Midi : Pique-nique au bord du lac ou resto sur place.
🌙 Soir : PAS de club. Dodo Grenoble en F1/hôtel. Demain rando Vanoise.
✅ Zéro activité obligatoire. Juste profiter. Repos total avant la montagne.
J7 Vanoise — Rando Glaciers
🛣️ Route : Grenoble → Aussois (1h30).
🥾 Journée : Randonnée des Glaciers de la Vanoise depuis le refuge du Fond d'Aussois. Sentier balisé, pas technique, vue sur les glaciers et la Dent Parrachée (3697m). 4-5h aller-retour, accessible.
🎉 Soir : Le Corsaire à Chambéry — bar rooftop 4.8★/472 avis, bières, tapas. CO2 Club n'existe pas → remplacé.
✅ Rando gratuite, juste de bonnes chaussures. Pas de matos d'escalade.
J8 🏊 Annecy — Retour
🛣️ Route : Chambéry → Annecy (40 min).
🏊🛶 Matin : Lac d'Annecy — dernière baignade, paddle, farniente. L'eau la plus pure d'Europe.
🚗 Aprem : Retour Annecy → Bonnelles (~550 km, 5h30). Arrivée dans la soirée.
✅ Fin du trip. Vous avez survécu au Diable. Respect.
