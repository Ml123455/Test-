🗺️ Ardèche Express — 3.5 Jours

2 personnes · Honda Jazz · Départ Bonnelles · Dimanche → Mercredi

~1140 km · ~210-260€/pers · Via Ferrata + Kayak + Canoë

J1 🌊 Dim — Via Ferrata Pont du Diable
🚗 Matin : Départ 6h de Bonnelles. ~570 km via A6/A7/A75 (5h30). Arrivée 11h30 à Thueyts.
🧗 Aprem : Via Ferrata du Pont du Diable (AD, 2h). Tyrolienne 90m au-dessus de l'Ardèche. Location matos 13€ Cimes-Canyons sur place.
🏊 Fin aprem : Baignade dans l'Ardèche sous le pont. Eau fraîche, parfaite après l'effort.
🌙 Nuit : Bivouac forêt Thueyts (gratuit).
✅ Ferrata accessible dès 1m40/45kg. Pas besoin d'être expert.
J2 🛶 Lun — Kayak Chassezac + Pont d'Arc
🚗 Route : Thueyts → Chassezac (45 min, D104).
🛶 Matin : Kayak sur le Chassezac — gorges sauvages, eau turquoise, plages secrètes. 2-3h, débutants OK. ~20€/pers (location kayak + gilet).
🏊 Aprem : Pont d'Arc — baignade, plage de sable. LE spot carte postale de l'Ardèche.
🌙 Nuit : Bivouac Vallon Pont d'Arc (gratuit).
✅ Kayak accessible à tous. Gilet fourni. Chassezac = moins de monde que l'Ardèche.
J3 🛶 Mar — Descente de l'Ardèche
🛶 Journée : Descente canoë de l'Ardèche — parcours Vallon → Saint-Martin (8 km, 2-3h) ou Vallon → Sauze (24 km, journée). Location ~20€/pers. Baignade au fil de l'eau.
🍽️ Midi : Pique-nique sur une plage au bord de l'eau.
🏊 Aprem : Baignade, chill. Dernière soirée dans le sud.
🌙 Nuit : Bivouac (gratuit) ou camping municipal (~10€) pour douche avant retour.
✅ Canoë accessible à tous. Gilets fournis. Parcours adapté débutants.
