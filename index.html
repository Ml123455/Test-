Questions fréquentes

Comment le bot connaît-il mes documents ?
Vous les importez en PDF, Markdown ou TXT depuis votre dashboard. Le bot les indexe automatiquement (embeddings Mistral). Quand un membre pose une question, il cherche dans vos documents et répond avec les sources.
Je dois savoir coder ?
Zéro code. Vous importez vos documents, vous invitez le bot via un lien Discord, c'est fait. Configuration depuis le dashboard.
Où sont stockées mes données ?
En France exclusivement (OVHcloud). Les embeddings sont calculés par Mistral (Paris). Rien ne transite par les États-Unis. Le jour où vous partez, tout est supprimé.
Et si j'ai 200 employés sur mon serveur ?
Le plan Business (69,99€/mois, 50 000 questions) est fait pour vous. Dépassement automatique, plafond de dépense paramétrable. Pour les grands comptes, plan Conformité à 149,99€/mois.
L'essai gratuit est vraiment sans CB ?
Oui. 7 jours, accès complet, zéro carte bancaire. Si ça ne vous convient pas, on supprime tout.
